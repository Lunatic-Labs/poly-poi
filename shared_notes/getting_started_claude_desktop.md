# Getting Started: Using Claude Desktop (Cowork) to Set Up the Project

This is an alternative to the manual Git setup guide. Using Claude Desktop's **Cowork** tab, Claude can handle the setup steps for you — no terminal commands required.

> **Note:** Cowork requires a **paid Claude subscription** (Pro, Max, Teams, or Enterprise). Check [claude.ai/pricing](https://claude.ai/pricing) if you're unsure what plan you have.

> **Free alternative:** If you don't have a paid Claude subscription, check out [OpenCode](https://opencode.ai/) — a free, open-source AI coding assistant with a desktop app for Mac, Windows, and Linux. It supports multiple AI models including Claude, GPT, and Gemini.

---

## Step 1: Install Claude Desktop

1. Go to [claude.ai/download](https://claude.ai/download)
2. Download the version for your operating system:
   - **Mac:** Click the macOS download link
   - **Windows:** Click the Windows download link
3. Run the installer and follow the prompts
4. Launch the app — it will appear in your Applications folder (Mac) or Start menu (Windows)

---

## Step 2: Sign In

Sign in with your Claude account. If prompted to verify in a browser, do so and then restart the app.

---

## Step 3: Open the Cowork Tab

1. Open Claude Desktop
2. Click the **Cowork** tab at the top of the window

Cowork runs directly on your computer and can manage files on your behalf — no need to open a terminal or type any commands yourself.

---

## Step 4: Ask Claude to Download the Project

In the prompt box, type the following and press Enter:

```
Clone this GitHub repository to my Desktop: https://github.com/Lunatic-Labs/poly-poi
```

Claude will handle everything. It may ask permission before taking actions — click **Allow** or **Accept** to approve. When it's done, a `poly-poi` folder will appear on your Desktop with all the project files inside.

---

## Keeping Your Files Up to Date

Whenever teammates push new changes, open Claude Desktop, switch to the **Cowork** tab, and type:

```
Pull the latest changes from GitHub for the poly-poi project on my Desktop
```

Claude will sync your local copy for you.

---

## Exploring the Project with Claude

Once the project is set up, you can ask Claude things like:

- _"What files are in the poly-poi project and what do they do?"_
- _"Open the shared_notes folder and summarize what's there"_
- _"Help me understand this document"_

Claude has direct access to your local files and can read, explain, and help you work with them — no technical experience required.
